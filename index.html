Home Page Person Form Transaction Form

I've created two forms for this exercise.

The first form takes in a person's name, age and address, as well as any of their childrens' names and age. This was built out of a piece of code which had editable tables, I've used to add the ability to add multiple children for a single person.

The second form creates a transaction record with the following fields:

  • Time
  • Account number
  • Amount

Both of these forms have a save button which saves the current state of the form and stores it in local storage.
